# Roulette Game

This is a simple roulette web application with a membership system built using Node.js, Express, EJS, and SQLite.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm start
   ```
3. Open `http://localhost:3000` in your browser.

## Features
- User registration and login using hashed passwords.
- Simple roulette game with red/black/green bets.
- Flash messages for errors and results.
